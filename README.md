# Mongoose-Express Rest API

Purpose
> Have an easy to plug rest api for entire mongodb db when using mongoose and express

##Insturctions
###get the code
```sh
$ cd [path-to-route-folder]
$ git clone [git-repo-url] .
```
####Add to your app.js:
```js
require('routes/mongoose-express-rest-api')(app);
```


#Enjoy
